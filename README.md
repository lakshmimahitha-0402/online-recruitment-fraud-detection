# online-recruitment-fraud-detection

## Description
This project detects fraudulent job postings using Python, NLP, and Deep Learning.
It helps job seekers identify fake recruitment offers by analyzing job descriptions.

## Technologies Used
- Python
- Flask
- SQLite
- NLP
- Deep Learning (ALBERT)
- HTML, CSS

## Features
- Fraud vs Genuine job classification
- Text preprocessing and NLP pipeline
- REST API using Flask
- SQLite database integration

## How to Run
1. Clone the repository
2. Install required dependencies
3. Run the Flask application
4. Access the app via browser

## Note
The trained model and dataset are not included due to large file size.
